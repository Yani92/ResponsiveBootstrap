Manual de usuarios Sistema - Phoenix

Bienvenidos al manual de usuarios del sistema "Phoenix" creado por estudiantes de la carrera de Desarrollo de Software, con la finalidad de satisfacer las necesidades presentadas por los docentes y autoridades del Instituto Superior Tecnológico del Azuay.
A través de este manual usted podrá visualizar como funciona nuestra aplicación y para que sirve cada opción proporcionada.
Antes de avanzar, usted, se preguntará ¿De qué se trata esta aplicación? Básicamente se trata de facilitar las labores de docentes, personal administrativo, autoridades, etc. A través de un proceso de matricula más sencillo, asistencia, ingreso de personas (alumnos, docentes), registro de cursos, anulación de matrículas, ingreso de notas, registro de asistencia, reportes de todo lo antes mencionado y muchas opciones más.

Objetivo General


El objetivo general de este proyecto es el desarrollo de una aplicación para dispositivos móviles el cual permitirá de forma más cómoda y rápida el acceso a la información de las notas de los estudiantes. Facilitando asi, cualquier inconveniente o consulta que desee realizar el estudiante.

Objetivos Específicos


- Desarrollar una aplicación móvil que permita consultar las notas meidante un previo registro del usuario.
- Mostrar información de las notas del ciclo mediante los datos guardados en la base de datos del sistema.
